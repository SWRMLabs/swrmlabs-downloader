# Light Client Go implementation

![swrmlabs](https://assets.swrmlabs.io/images/logo-icon.svg)


### Usage

```
Usage:
./client <OPTIONS>

Options:
		
  -bin string
    	Location of lite-client (default "ss-light-linux-amd64")
  -sharable string
    	Sharable string provided for file

```
